# KlikBCA

## Cara pakai :
1. Edit cek_saldo.php
2. Ubah dengan username dan password klikbca anda.
```php
$_p['value(user_id)'] = "username"; // username
$_p['value(pswd)'] = "password anda"; // password
```
3. Jalankan cek_saldo.php via CLI atau browser.
